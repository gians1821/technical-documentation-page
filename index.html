What is Linux?

Linux is a family of open-source Unix-like operating systems based on the Linux kernel, an operating system kernel developed by Linus Torvalds in 1991. The Linux kernel serves as the foundation for various operating systems, commonly referred to as Linux distributions or simply "distros."

These distributions typically include a collection of software applications, utilities, libraries, and graphical interfaces to provide a complete operating environment.

Here are some examples of distros:

  • Debian
  • Ubuntu
  • Kali
  • Arch
What is a terminal?

A terminal typically refers to a text-based interface used to interact with a computer system. It's a command-line interface (CLI) where users can input commands to perform various tasks such as navigating the file system, running programs, managing files, and configuring settings.

Terminals are often found in Unix-based operating systems like Linux and macOS, although Windows also has its own command-line interface known as Command Prompt or PowerShell.

What is a shell?

A shell is a user interface for accessing an operating system's services. It's a program that takes commands from the user and translates them into instructions that the operating system can understand and execute.

Shells can be command-line interfaces (CLI), where users type commands using text, or graphical user interfaces (GUI), where users interact with the shell using graphical elements like buttons and menus.

The most common shells are the next ones:

  • Bash
  • Zsh
  • Powershell
What is Bash?

Bash, short for "Bourne Again Shell," is a command-line interpreter and scripting language primarily used in Unix and Unix-like operating systems, including Linux and macOS.

Bash is highly customizable and extensible, offering features such as command-line editing, history support, tab completion, and the ability to define aliases and functions.

What are the most used commands?

The most used commands on Linux can vary depending on the user's specific tasks and workflows. However, some of the commonly used commands across various contexts include:

  • cd

    Change directory

  • ls

    List directory contents

  • pwd

    Print working directory

  • mkdir

    Make directory

  • rm

    Remove files or directories
